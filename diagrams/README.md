## diagrams

Architecture diagrams stored as Excalidraw files.

### Files

- \*.excalidraw: editable diagrams (system design, data flow, permissions, SSE, etc.).

### How to view/edit

- Open the .excalidraw files in VS Code with an Excalidraw extension, or
- Upload them to https://excalidraw.com to view/edit.

### Exporting

When you update a diagram, export a PNG/SVG for sharing and keep the source
.excalidraw file in this folder.
