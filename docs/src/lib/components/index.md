# components

All files in here are .svelte files that are to be used like helper functions in the main application. They can be imported and used in other components or pages and should NEVER have any side effects. They are purely presentational and should not contain any logic that passively alters the state of the application without user interaction.


This directory contains:

- [account](account/index.md)
- [bootstrap](bootstrap/index.md)
- [dashboard](dashboard/index.md)
- [forms](forms/index.md)
- [general](general/index.md)
- [permissions](permissions/index.md)
