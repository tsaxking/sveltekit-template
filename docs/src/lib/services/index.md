# services
A front-end service is a program that runs alongside the main application, providing additional functionality or features. These services can be used to handle tasks such as data fetching, state management, or real-time communication.

I am planning to integrate a chat service into the application, which will allow users to communicate in real-time. This service will be built using sse and will enable users to send and receive messages instantly.
