# src/lib/utils/ag-grid

Ag Grid is a fantastic resource for building high-performance data grids in web applications. This directory contains utility components that expand the functionality of Ag Grid, making it easier to integrate and customize within your projects.

## Contents

- `buttons`: Utility functions for creating and managing buttons within Ag Grid cells.
- `date-time`: Utilizes flatpickr to provide date and time pickers for Ag Grid cells.
- `search-select`: Implements searchable dropdowns for Ag Grid cells, enhancing user experience when selecting options from large datasets.
- `checkbox-select`: Provides multi-select checkboxes for Ag Grid cells, allowing users to select multiple rows easily.
